# Changelog

All notable changes to EGXpilot will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0-beta.1] - 2025-06-18

### Added
- Initial project structure
- Authentication system with JWT
- Stock data collection and management
- Technical analysis service
- Portfolio management
- Opportunity hunter (temporarily disabled)
- Redis caching support
- SQL Server database integration
- REST API endpoints
- WebSocket support for real-time updates
- Docker configuration
- Health check endpoints

### Changed
- Organized environment configurations
- Improved error handling
- Updated project structure

### Fixed
- Database connection issues
- Redis configuration
- Environment variable loading

### Security
- Added proper authentication middleware
- Implemented rate limiting
- Added input validation
